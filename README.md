# 鋒兄Next資訊管理系統

一個基於 SwiftUI 的 macOS 綜合管理應用程式，提供多種實用的管理功能。

## 功能特色

### 🏠 儀表板
- 系統概覽和統計數據
- 快速操作面板
- 重要提醒和警告

### 🖼️ 圖片展示
- 圖片庫管理
- 支援多種格式
- 搜尋和分類功能

### 📅 訂閱管理
- 訂閱服務追蹤
- 到期提醒
- 費用統計
- 自動續費管理

### 🍎 食品管理
- 食品庫存管理
- 過期日期追蹤
- 數量控制
- 分類管理

### 🎬 影片介紹
- 影片播放和管理
- 快取功能
- 儲存空間管理

### 🎵 音樂歌詞
- 歌曲庫管理
- 多語言歌詞支援
- 收藏功能

### ℹ️ 關於我們
- 公司資訊
- 聯絡方式
- 服務介紹

## 系統要求

- macOS 13.0 或更高版本
- Xcode 15.0 或更高版本
- Swift 5.9 或更高版本

## 安裝說明

1. 克隆此專案到本地
```bash
git clone [repository-url]
cd macosappkiro20251230
```

2. 使用 Xcode 開啟專案
```bash
open macosappkiro20251230.xcodeproj
```

3. 選擇目標設備並運行

## 專案結構

```
macosappkiro20251230/
├── macosappkiro20251230App.swift    # 應用程式入口
├── ContentView.swift                # 主視圖和導航
├── Views/                          # 各功能模組視圖
│   ├── DashboardView.swift         # 儀表板
│   ├── GalleryView.swift           # 圖片展示
│   ├── SubscriptionView.swift      # 訂閱管理
│   ├── FoodManagementView.swift    # 食品管理
│   ├── VideoView.swift             # 影片介紹
│   ├── MusicView.swift             # 音樂歌詞
│   └── AboutView.swift             # 關於我們
└── Assets.xcassets/                # 資源文件
```

## 主要特性

- **響應式設計**: 適配不同螢幕尺寸
- **模組化架構**: 清晰的代碼結構
- **現代化 UI**: 使用 SwiftUI 最新特性
- **資料管理**: 本地資料存儲和管理
- **用戶體驗**: 直觀的操作界面
