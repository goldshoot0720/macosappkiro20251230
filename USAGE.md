# 使用指南

## 快速開始

### 1. 啟動應用程式
- 在 Xcode 中開啟專案
- 選擇 macOS 目標設備
- 點擊運行按鈕或按 `Cmd+R`

### 2. 導航使用
- 左側邊欄包含所有功能模組
- 點擊任意項目切換到對應功能
- 藍色高亮顯示當前選中的模組

## 各模組使用說明

### 儀表板 📊
**功能概覽**
- 查看系統整體統計
- 監控重要指標
- 快速訪問常用功能

**主要元素**
- 統計卡片：顯示關鍵數據
- 管理統計：食品和訂閱狀態
- 快速操作：一鍵訪問功能
- 警告提醒：重要事項通知

### 圖片展示 🖼️
**功能說明**
- 管理和展示圖片資源
- 支援網格視圖瀏覽
- 搜尋和篩選功能

**操作方式**
- 點擊「新增圖片」添加新內容
- 使用搜尋框快速查找
- 點擊圖片查看詳細資訊

### 訂閱管理 📅
**核心功能**
- 追蹤所有訂閱服務
- 監控到期日期
- 計算總費用

**使用步驟**
1. 填寫服務名稱
2. 輸入訂閱 URL（可選）
3. 設定費用金額
4. 選擇到期日期
5. 點擊「新增訂閱」

**狀態說明**
- 🟢 正常：距離到期超過7天
- 🟡 注意：3-7天內到期
- 🔴 緊急：3天內到期或已過期

### 食品管理 🍎
**主要用途**
- 管理食品庫存
- 追蹤保存期限
- 控制數量變化

**操作指南**
1. 輸入食品名稱
2. 設定數量
3. 選擇過期日期
4. 添加圖片 URL（可選）
5. 點擊「新增食品」

**數量調整**
- 使用 + / - 按鈕調整數量
- 即時更新庫存狀態

### 影片介紹 🎬
**功能特色**
- 播放和管理影片
- 本地快取功能
- 儲存空間監控

**使用方法**
- 點擊「播放影片」觀看內容
- 使用「快取」功能離線觀看
- 監控儲存使用情況

### 音樂歌詞 🎵
**核心功能**
- 管理歌曲庫
- 多語言歌詞支援
- 收藏喜愛歌曲

**操作說明**
- 使用搜尋框查找歌曲
- 點擊愛心圖標收藏
- 點擊「查看歌詞」閱讀內容

### 關於我們 ℹ️
**內容包含**
- 公司介紹和使命
- 團隊成員資訊
- 聯絡方式
- 服務特色說明

## 快捷鍵

- `Cmd+1` - 儀表板
- `Cmd+2` - 圖片展示
- `Cmd+3` - 訂閱管理
- `Cmd+4` - 食品管理
- `Cmd+5` - 影片介紹
- `Cmd+6` - 音樂歌詞
- `Cmd+7` - 關於我們

## 常見問題

**Q: 如何備份資料？**
A: 目前版本使用本地存儲，建議定期備份應用程式資料夾。

**Q: 支援哪些圖片格式？**
A: 支援 JPEG、PNG、GIF 等常見格式。

**Q: 如何匯出資料？**
A: 可以通過各模組的匯出功能將資料保存為 CSV 或 JSON 格式。

**Q: 應用程式卡頓怎麼辦？**
A: 嘗試重啟應用程式，或清理快取資料。
